# SAFE

Installation
------------

Prerequisites:

Node

MongoDb

Bower

Clone the repository:
```
git clone git@github.com:ApolloRobotics/SAFE.git
```
Install dependencies:
```
npm install
```
Initiate task runner:
```
gulp
```

Pulling Changes
------------

Pull the repository:
```
git pull
```
Initiate task runner:
```
gulp
```

Pushing Changes
------------
```
git commit -am "Example Comment [#SAFE-Num]"
git pull
git push
```
